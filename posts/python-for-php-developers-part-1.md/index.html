How does Python differ from PHP?

Variables in Python vs PHP

Unlike php there is no special characters to signifies a variable in python,

x, y= 10, 100

Code block

To distinguish code blocks, PHP like any classic programming language depends on braces, more precisely, curly braces, that’s not case when it comes to python, python hinges on indention.

i = 1
while i < 6:
  print(i)
  i + = 1

Conditional

There is no significant difference between python and php in terms of conditional, may be the only subtle difference of elif in python is the equivalent of else if in php.

Expression can precede condition

Python doesn’t support same ternary operator not exactly but not far off being concise

a if condition else b

st = "x is less than y" if  (x<y) else "x is greater than or the same as y"

Loop

 while(x < 5):
    print(x)
    x = x+1

Python doesn’t have do-while loop, it supports only while and for loop, speaking of for loop, python doesn’t have the traditional form of for loop for(x = 0 ; x > 5 ; x++), there is no loop counter like you see in javascript, for in python is iterative

1
2

for x in range(5,10)
    print(x)

However if you have a use case that requires a counter, the enumerate() function lend itself to this purpose, as it adds a counter to the iterable and returns it in a form of enumerate object.

1
2
3

days = ["MON", "TUE", "WED", "THU", "FRI"]
    for i, d in enumerate(days):
        print(i, d)

The output will be

0 MON
1 TUE
2 WED
3 THU
4 FRI

Function in python vs php

Function is defined by the keyword ‘def’ in python whereas it’s function in php
